Description of Service

VaultSync is a free, open-source plugin that synchronizes your Obsidian vault files with cloud storage services. All sync operations occur directly between your local device and your cloud storage account. The Plugin does not operate any intermediary servers.

User Responsibilities

  • Cloud Account Setup: You are responsible for creating and configuring your own Google Cloud Project, including obtaining OAuth credentials (Client ID / Client Secret). The Plugin does not provide shared credentials.
  • Data Backup: You are responsible for maintaining backups of your vault data. While the Plugin is designed for reliable synchronization, no software can guarantee zero data loss.
  • E2EE Password Management: If you enable End-to-End Encryption, you are solely responsible for remembering your encryption password and backing up the vault-lock.vault file. There is no password recovery mechanism. Lost passwords or vault-lock files will result in permanent, unrecoverable data loss.
  • Credential Security: You must keep your OAuth Client Secret and E2EE password confidential. Do not share them with others.
